# Demian Max | Master Edition v5

Esta é a versão definitiva, otimizada para responsividade, contraste e facilidade de edição.

## 📸 Guia de Imagens (Fácil Edição)
Para trocar qualquer imagem do site, basta substituir os arquivos na pasta `assets/img/` seguindo os nomes abaixo:

- **Hero Principal:** `assets/img/hero/hero-principal.jpg`
- **Logo Principal:** `assets/img/logos/logo.png`
- **Foto Experiência:** `assets/img/gallery/experiencia-1.jpg`
- **Foto Mídia 1:** `assets/img/gallery/midia-1.jpg`
- **Foto Mídia 2:** `assets/img/gallery/midia-2.jpg`
- **Logos de Clientes:** `assets/img/gallery/clientes-logos.jpg`
- **Foto Sobre:** `assets/img/gallery/sobre-1.jpg`

**Dica de Webmaster:** Para manter o visual perfeito, tente usar fotos com boa resolução. O site está configurado para **não cortar** suas fotos e manter as cores originais (sem filtros).

## 🛠️ Melhorias Técnicas (Webmaster Level)
1. **Responsividade:** Menu simplificado que não quebra em duas linhas. No celular, um menu inteligente em tela cheia é ativado.
2. **Scroll Spy:** Ao clicar em um link do menu, o site desliza suavemente e para exatamente no início da seção, sem esconder o título atrás da barra.
3. **Contraste:** O formulário de briefing agora possui contraste máximo (texto branco em fundo escuro suave) para garantir legibilidade em qualquer tela.
4. **Fidelidade:** Removidos todos os filtros de imagem. Suas fotos aparecem exatamente como são.

## 🚀 Como Publicar
1. Suba todos os arquivos para a raiz do seu repositório no GitHub.
2. Ative o GitHub Pages nas configurações do repositório.
